What is Kuptm e-Booking Venue page content?


This application use for user to book venue at Kuptm that available in this application.

How can user book a venue?
User need to click on the " START BOOKING " button to proceed booking process at the home page. Then, user need to fill up all the requirement and submit the booking by click " BOOK NOW ".

How can user edit/cancel the booking?
User can click on booking icon then it will going to booking history section. If user would like to edit the booking, user need to slide to the right and click on pencil icon and it will bring to edit form and if user has confirm the edit just click on update button below. If user want to cancel, user need to slide to the left then click on delete icon.
